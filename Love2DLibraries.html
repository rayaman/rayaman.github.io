GuiManager

The GuiManager library was created with the intent to provide an easy way to implement menus and other visual effects into ones game. This is not a GUI library like LoveFrames! It doesn't provide premade menus for the most part. However this library makes it easy to create your own menus. Check the example page for examples of how one could go about using this library. I suggest reading the documentation in order: MultiManager (Very important Library to understand), LoveBind(Not as important but a list of features that are changed are shown), After this take a look at GuiManager Syntax/Core, 

MultiManager/LoveBind

This is a multitasking library that I have been working on, on and off, for 3 years. It went through 3 complete rewrites and is 100% compataible with  all versions of lua 5.1+. This library can be used in any version of lua that has a working os.clock(), lovebind allows the libraries features to use a bit of loves extras, It also uses its own custom run function.
